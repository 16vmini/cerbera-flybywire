Throttle map — pedal % → throttle %

5 break-points, linear between. Drag points up/down/left/right. The orange dot shows where the pedal currently is on the curve. Default = softer first 20 % (calmer in traffic), linear beyond. Apply to send to Pico, Save to flash to persist.

PID gains
